# Restaurant-Website
A full-stack restaurant website built using PHP and MySQL that allows users to browse menu items, make reservations, and interact with a dynamic database-driven system. The project includes an admin panel for managing menu items and demonstrates CRUD operations

Features:

Dynamic menu from database
Table reservation system
Admin login panel

Tech Stack:

PHP
MySQL
HTML/CSS
JavaScript
